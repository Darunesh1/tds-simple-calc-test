# Simple Calculator Test

## Overview
This is a simple calculator that adds 100 + 150 + 200 and displays the result (450) inside #total-sales. The page title is set to Sales Summary 2025, and Bootstrap 5 is used for styling.

## Features
- Adds 100 + 150 + 200
- Displays the result in #total-sales

## Usage
To use this application:
1. Clone or download the repository.
2. Navigate to the project directory.
3. Run `npm install` to install dependencies.
4. Start the server with `npm start`.

## Technical Details
- Uses HTML, CSS, and JavaScript for the front-end.
- Bootstrap 5 is used for styling.
